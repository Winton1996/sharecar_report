源代码：
* 代码仅供学习之用
* db包含了导入数据和分析数据所需要的脚本
* report包含了在线网站源代码

如何运行：
## 导入数据
* 准备好postgres数据库
* 安装必要的pip3包
* 执行import.py

## 查看报告
* npm install live-server -g 
* cd report
* live-server

* 数据没有提供，请自行爬取
